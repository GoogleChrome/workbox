workbox-background-sync Demo

  1. Open DevTools.
  2. Switch to the Network panel.
  3. Toggle `Disable cache` checkbox on (it's unchecked by default).
  4. Click `Make Fetch request` button below.
  5. Observe that request is successfully completed with status 200.
  6. Disconnect from actual network (DevTools offline doesn't work for background sync).
  7. Again, click `Make Fetch request` button below.
  8. Observe that request is aborted due to no network.
  9. Connect to your network and observe the Network panel in DevTools.
  10. Once the network is connected, you will see that the request is retried automatically upon the `sync` event.
